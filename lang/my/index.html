အဓိက အခြေခံအဆောက်အဦ စီမံကိန်းများတွင် ကျယ်ပြန့်သော အတွေ့အကြုံရှိသည့် ထူးချွန်သော အမှုဆောင် မန်နေဂျာနှင့် မြို့ပြ အင်ဂျင်နီယာတစ်ဦးဖြစ်ပြီး ဒေတာ သိပ္ပံနှင့် ခွဲခြမ်းစိတ်ဖြာမှု အတိုင်ပင်ခံလုပ်ငန်းတွင် ကျွမ်းကျင်မှုဖြင့် ဖြည့်စွက်ထားပါသည်။ ကျွန်ုပ်သည် ကြီးမားသော စီမံကိန်းများကို အကောင်အထည်ဖော်ရန်နှင့် မဟာဗျူဟာမြောက် အတိုင်ပင်ခံ ဝန်ဆောင်မှုများ ပေးအပ်ရန် အင်ဂျင်နီယာဆိုင်ရာ အသိပညာကို စီမံခန့်ခွဲမှု ကျွမ်းကျင်မှုနှင့် ခွဲခြမ်းစိတ်ဖြာမှု ချဉ်းကပ်နည်းများဖြင့် ပေါင်းစပ်ထားပါသည်။


ကျွန်ုပ်အကြောင်း

ကျွန်ုပ်သည် နျူကလီးယားစွမ်းအင်၊ စက်မှုလုပ်ငန်းသုံး အဆောက်အအုံများနှင့် မြို့ပြ အခြေခံအဆောက်အဦများကို လွှမ်းခြုံသည့် အတွေ့အကြုံနှင့်အတူ ဆောက်လုပ်ရေး အင်ဂျင်နီယာနှင့် စီမံကိန်း စီမံခန့်ခွဲမှုတွင် ၂၁ နှစ်ကျော် လုပ်ကိုင်ခဲ့ပါသည်။ ကျွန်ုပ်၏ အလုပ်အကိုင် သက်တမ်းတစ်လျှောက်တွင် ဒီဇိုင်း၊ စီမံကိန်းရေးဆွဲခြင်း၊ စီမံကိန်း စီမံခန့်ခွဲမှု၊ စွန့်စားမှု ဆန်းစစ်ခြင်း၊ R&D နှင့် အင်ဂျင်နီယာဆိုင်ရာ IT ဖြေရှင်းချက်များတွင် အင်ဂျင်နီယာနှင့် စီမံခန့်ခွဲမှု ရာထူးနှစ်ခုလုံးကို ထမ်းဆောင်ခဲ့ပါသည်။

ကျွန်ုပ်သည် စီမံကိန်းနှင့် လုပ်ငန်းအဆင့် နှစ်ခုလုံးတွင် စွန့်စားမှုများကို စီမံခန့်ခွဲရာ၌ ကျွမ်းကျင်မှုနှင့်အတူ ဥပဒေနှင့် နည်းပညာဆိုင်ရာ ရှုထောင့်နှစ်ခုလုံးမှ EPC စီမံကိန်းများကို ပြည့်စုံစွာ နားလည်ပါသည်။ နျူကလီးယား ဓာတ်အားပေး စက်ရုံများတွင် နက်ရှိုင်းသော ကျွမ်းကျင်မှုရှိပါသည်။ ၂၀၁၆ ခုနှစ်မှစ၍ ဒေတာကို အခြေခံသော သုတေသနပြုလုပ်ရန် ကျွန်ုပ်၏ ဆောက်လုပ်ရေး ကျွမ်းကျင်မှုကို ဒေတာ သိပ္ပံနှင့် စက်သင်ယူမှု (Machine Learning) နည်းပညာများနှင့် ပေါင်းစပ်လုပ်ဆောင်လျက်ရှိပါသည်။ ကျွန်ုပ်၏ နည်းပညာဆိုင်ရာ ကျွမ်းကျင်မှုတွင် Data Science, Machine Learning, Python, Oracle Primavera, SQL နှင့် အမျိုးမျိုးသော ခွဲခြမ်းစိတ်ဖြာမှု နည်းပညာများ ပါဝင်ပါသည်။

ကျွန်ုပ်သည် အောက်ပါနယ်ပယ်များကို စိတ်ဝင်စားပါသည်:

  • နိုင်ငံတကာ ပူးပေါင်းဆောင်ရွက်မှုနှင့် နယ်စပ်ဖြတ်ကျော် စီမံကိန်းများ
  • မဟာဗျူဟာမြောက် စီမံခန့်ခွဲမှုနှင့် ဖွံ့ဖြိုးတိုးတက်ရေး
  • သုတေသန၊ ခွဲခြမ်းစိတ်ဖြာမှုနှင့် မဟာဗျူဟာမြောက် အတိုင်ပင်ခံလုပ်ငန်း
  • စီမံကိန်း စီမံခန့်ခွဲမှုနှင့် PM နည်းစနစ်
  • လုပ်ငန်းနှင့် စီမံကိန်း စွန့်စားစီမံခန့်ခွဲမှု
  • EPC စာချုပ် စီမံခန့်ခွဲမှုနှင့် ဥပဒေဆိုင်ရာ နည်းပညာ (Legal Tech)
  • နျူကလီးယားနှင့် ကြီးမားသော အခြေခံအဆောက်အဦ ဆောက်လုပ်ရေး
  • ဒေတာ သိပ္ပံ၊ စက်သင်ယူမှု (Machine Learning) နှင့် ဘာသာစကား မော်ဒယ်ကြီးများ (LLMs)
