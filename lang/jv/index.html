Babagan kula

Kula wis nyambut gawé ing teknik konstruksi lan manajemen proyek luwih saka 21 taun, kanthi pengalaman nyakup energi nuklir, fasilitas industri, lan infrastruktur sipil. Sadawaning karir kula, kula wis nyekel posisi teknik lan manajemen ing desain, perencanaan, manajemen proyek, analisis risiko, R&D, lan solusi IT kanggo teknik.

Kula nduwèni pangerten lengkap babagan proyek EPC saka perspektif hukum lan teknis, kanthi keahlian ngatur risiko ing tingkat proyek lan perusahaan. Kula nduwèni keahlian jero ing pembangkit listrik tenaga nuklir. Wiwit 2016, kula wis nggabungaké keahlian konstruksi kula karo teknik ilmu data lan machine learning kanggo nindakaké panalitèn adhedhasar data. Keahlian teknis kula kalebu kaprigelan ing Ilmu Data, Machine Learning, Python, Oracle Primavera, SQL, lan macem-macem teknologi analitis.

Kula kasengsem ing bidang-bidang ing ngisor iki:

  • Kerjasama internasional lan proyek lintas-wates
  • Manajemen lan pangembangan strategis
  • Panalitèn, analisis, lan konsultasi strategis
  • Manajemen proyek & metodologi PM
  • Manajemen risiko perusahaan lan proyek
  • Manajemen kontrak EPC lan Legal Tech
  • Konstruksi nuklir lan infrastruktur gedhé
  • Ilmu Data, Machine Learning lan Model basa gedhé (LLM)
